# Primary Data Types
 The primary (primitive) data types in Javascript are:
 * String
 * Number
 * Boolean


 The composite (reference) data types are:
 * Object
 * Array

 The special data types are:
 * Null
 * Undefined


# Unary Operators
You can find the data type of a value using the typeof operator.

    typeof "myString"
    =>"string"

    typeof 31
    =>"number"
